# Employee Polls App

## Description
This project has been made using React/Redux and TypeScript 

## Set up
1. Make sure Node and npm are installed from the terminal. Steps:

    ```
    npm install
    ```

2. Start the project.

    ```
    npm start
    ```

3. Now you can visit http://localhost:3000 to view the client

4. To run the Tests: 

    ```
    npm run test
    ```
